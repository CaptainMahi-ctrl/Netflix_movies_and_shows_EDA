# Netflix Movies & TV Shows — Exploratory Data Analysis

## Overview
Exploratory Data Analysis on 8,800+ Netflix titles to uncover content strategy, growth trends, and audience targeting insights.

## Tools Used
Python, Pandas, Matplotlib, Seaborn

## Key Insights
- Netflix catalog is ~70% Movies and ~30% TV Shows
- Content additions peaked in 2019, dipped post-2020 (COVID)
- Q4 (Oct–Dec) sees highest content additions — holiday strategy
- USA dominates production; India growing rapidly
- 60%+ of TV shows have only 1 season — limited series strategy
- TV-MA is the most common rating — targets mature audiences

## Dataset
Download from Kaggle: https://www.kaggle.com/datasets/shivamb/netflix-shows
